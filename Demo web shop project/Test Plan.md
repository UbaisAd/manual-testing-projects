# Test Plan – Demo Web Shop

## Project Name
Demo Web Shop Testing

## Prepared By
Ubais Ahamed



## Objective
To validate the core functionality of the Demo Web Shop application, including user authentication, product browsing, cart operations, wishlist management, and checkout process.

## Scope
- Authentication (Login / Register / Logout)
- Product Catalog
- Search
- Product Details Page
- Shopping Cart
- Wishlist
- Checkout / Order Process
- User Account

## Out of Scope
- Payment gateway integration
- Performance testing
- Security testing

## Test Approach
- Manual Testing
- Functional Testing
- Exploratory Testing

## Test Environment
- Browser: Google Chrome
- Operating System: Windows

## Modules
1. Authentication
2. Product Catalog
3. Search
4. Product Details Page
5. Shopping Cart
6. Wishlist
7. Checkout
8. User Account
